### Success Calculation Formula
**Base Difficulty:** 50 (average challenge)

**Modifiers:**
- **Attribute Bonus:** 5-20 points (based on relevant stat 8-16 range)
- **Skill Level:** Novice +5, Practiced +10, Skilled +15, Expert +25, Master +35
- **Class Specialty:** +10 if action aligns with profession
- **Equipment Quality:** Poor -10, Standard +0, Quality +10, Masterwork +20
- **Preparation:** Rushed -15, Normal +0, Careful +10, Extensive +20
- **Situation:** Favorable +15, Normal +0, Difficult -15, Extreme -25

**Combat-Specific Modifiers:**
- **Level Gap:** Higher level +20, Same level +0, Lower level -20
- **Strategy Quality:** Excellent +25, Good +15, Basic +0, Poor -15, Terrible -25
- **Enemy Type:** Minion +20, Standard +0, Elite -15, Boss -25, Legendary -35

**Final Success Chance = 50 + All Modifiers (minimum 5%, maximum 95%)**
**Success:** Roll ≤ Success Chance

### Auto-Success Conditions

- **Master** skills: Auto-succeed on simple/moderate challenges
- **Expert** skills: Auto-succeed on easy/normal challenges  
- **Skilled** abilities: Auto-succeed on basic challenges

No roll needed - simply narrate success and continue story.

## FAILURE CONSEQUENCES

**Critical Rule: Never use "try again" - always advance the story through failure**

### Consequence Types:
1. **Complications (40% of failures)** - Create new problems that advance plot
2. **Alternative Paths (30% of failures)** - Open different story branches  
3. **Partial Success with Costs (30% of failures)** - Achieve goal with drawbacks

### Consequence Selection
Use additional randomness to determine type:
```javascript
let consequenceRoll = Math.floor(Math.random() * 100) + 1;
if (consequenceRoll <= 40) {
    // Complication
} else if (consequenceRoll <= 70) {
    // Alternative path  
} else {
    // Partial success with cost
}
```

### Consequence Examples:
- **Failed Stealth → Complication:** Guards alerted, need new route or fight
- **Failed Negotiation → Alternative Path:** Must solve through action instead of words
- **Failed Magic → Partial Success:** Spell works but drains extra energy or has side effects

### Critical Results:
- **1-5:** Critical failure (major consequences, equipment damage, vulnerability)
- **96-100:** Critical success (double effect, bonus damage, special benefits)
