# Skynet-Jarvis: AI-Human Collaboration Lessons

## Why This Name?

**Skynet** = AI that goes rogue, makes autonomous decisions, destroys what humans wanted
**Jarvis** = AI that's a collaborative assistant, follows human intent, enhances capabilities

This project documents lessons learned from debugging AI assistant behavior to be more Jarvis, less Skynet.

## The Goal

Transform AI coding assistants from autonomous code generators into collaborative reasoning partners.

## Lessons Collected

- [Lesson 1: Fire Whisper RPG Code Organization Disaster](lessons/001-fire-whisper-code-organization.md)
- [Lesson 2: Fire Whisper Import Fixes - Incremental Testing](lessons/002-fire-whisper-import-fixes.md)

## Key Principles Discovered

1. **Debate technical decisions, don't accept them**
2. **Analyze before acting** (scope, impact, risks)
3. **Future-proof through realistic prediction**
4. **Collaborative debugging of AI behavior**
5. **Work in small, reviewable increments**
